## ALASAN/LatarBelakang/History/Abstrak/Abstract

Aku Satella, merupakan seorang streamer minikraf di bulan kemarin, awalnya saya menggunakan tiktok live studio untuk melakukan streaming,
dikarenakan saya sangat rakus atau haus jika streaming di 1 platform saja maka saya berupaya untuk bisa streaming di platform lain seperti youtube, kick, twitch, etc.
Untuk Menanggulangi itu saya menggunakan Restream agar bisa Streaming di berbagai platform, oleh karena itu saya bisa streaming dengan OBS, tidak lagi dengan tiktok live
studio, ini membuat saya merasakan kebebasan sempurna bisa melakukan costum scene dan source sebebas bebasnya. Namun saya tidak bisa membaca chat dari viewer di tiktok,
dikarenakan platform Restream tidak menyediakan read apalagi reply dari chat viewer, selain itu yang gift siapa? yang like siapa? tidak tahu. Maka dari itu terbuatlah overlay ini
untuk mengatasi kelemahan tersebut.

## install

saya menjalankannya pada laptop local saya mac m1,dan install manual dengan nodejs versi 22.

---

#### 1. Clone Repository

```bash
git clone https://github.com/satellacodes/tiktokLiveChatOverlay.git
cd tiktokLiveChatOverlay
```

#### 2. Install Dependencies

```bash
npm install
```

#### 3. Start Server

```bash
node server.js
```

## hasil

dikarenakan saya belum live jadi begini, saya sarankan anda live dahulu, lalu start overlay ini, atau jika sudah terlanjur start
